# cross-chain-nft-marketplace
A one-click, cross-chain marketplace for NFT's. Buy any NFT on any EVM-based chain with any EVM token.


## Main components

ETH contract <--> Uniswap <--> Stargate <--> Trader Joe <--> AVAX contract
